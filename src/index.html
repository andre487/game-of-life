Help

Basics

This is Conway’s Game of Life.

You can place figures to the field by mouse. A lot of patterns are described on Wikipedia.

Controls

When you have set up the map, you can start the game with Start button.

You can stop the game any time with Stop button and make changes on the map.

You can save and load game with Save and Load buttons. Game saves to localStorage.

Reset button clears the map.

Navigation

When mouse cursor is over the field, you can navigate on field by mouse wheel or arrow keys or WASD keys.

You can zoom field by Ctrl+Wheel or Cmd+Wheel. Or by arrow keys or WASD with Ctrl or Cmd.
